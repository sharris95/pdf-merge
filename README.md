# PDF Merge Tool

Merge multiple PDF files locally in your browser. Drag to reorder, then merge and download.

![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-live-brightgreen?logo=github)
![Vercel](https://img.shields.io/badge/Vercel-live-black?logo=vercel)

## Features

- Drag-and-drop PDF upload and reordering
- Local, in-browser merge (no server uploads)
- One-click download of the merged PDF

## Live Sites

- GitHub Pages: https://sharris95.github.io/pdf-merge/
- Vercel: https://pdf-merge-lime.vercel.app/

## Development

```bash
npm install
npm run dev
```
